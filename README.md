Superstore Retail Performance Analysis (Power BI Dashboard)

This project focuses on analysing retail sales data using Power BI to generate insights on sales, profit, and business performance. The dashboard helps in understanding trends, identifying top-performing areas, and supporting data-driven decisions.

📊 Key Features:
Interactive dashboard with filters for region (Central, East, South, West)
KPI tracking: Sales, Profit, Quantity, Returns, Delivery Days
Sales analysis by category, segment, and payment mode
Top-performing states based on sales and profit
Monthly sales and profit trend (Year-over-Year analysis)
Shipping mode and sub-category performance insights

🛠 Tools & Technologies
Power BI
Excel (Data Source)
Power Query (Data Cleaning)
DAX (Calculated Measures)

📈 Insights Generated
Identified high-performing states like California and New York
Found key revenue-driving product categories
Analysed seasonal trends in sales and profit
Evaluated shipping methods impacting delivery and performance

🎯 Purpose

The goal of this project is to demonstrate data analysis and visualization skills using Power BI and to show how dashboards can support business decision-making.
